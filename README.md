# WikiParse
Тестовое задание от центра МАСТ

Помимо википедии, можно выделить еще пару подходов к отслеживанию информации о смертях.
1) Мониторинг новостных пабликов и СМИ:
  - Использование API новостных платформ
  - Написание парсеров
2) Мониторинг соц. сетей (декомпозиция такая же, как и в первом пункте списка)
3) По моему предположению должны существовать специализированные базы данных, в которых содержится актуальная информация о погибших. (скорее всего доступ к ним ограничен)

PS: Для запуска скрипта необходимо:
1) Установить все зависимости командой pip install -r requirements.txt
2) Изменить конфигурационные переменные в файле main.py
3) Запустить файл main.py
